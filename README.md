## Featured project: Stims

**Stims** is a collection of small, friendly webtoys built to support sensory engagement and self-regulation. The project emphasizes keyboard access, inclusive design, and playful experimentation.

- **What it is:** A library of accessible webtoys for neurodiverse users.
- **Focus:** Sensory-friendly interactions, low cognitive load, and gentle animation.
- **Tech:** HTML5, CSS3, JavaScript.
- **Explore:** <https://github.com/zz-plant/stims>
